# Dental Images Directory

This directory is for storing dental clinic-related images.

## Recommended Images

You can add the following types of images here:

1. **Hero Images** (`hero-dental-clinic.jpg`)
   - Modern dental clinic interior
   - Professional dental equipment
   - Clean, welcoming environment

2. **Service Images** (`service-*.jpg`)
   - Preventive care images
   - Restorative procedures
   - Cosmetic dentistry
   - Oral surgery

3. **Team Images** (`dentist-*.jpg`)
   - Professional dentist photos
   - Staff photos

4. **Facility Images** (`facility-*.jpg`)
   - Reception area
   - Treatment rooms
   - Waiting area

## Image Sources

You can use free images from:
- Unsplash (https://unsplash.com/s/photos/dental-clinic)
- Pexels (https://www.pexels.com/search/dental/)
- Pixabay (https://pixabay.com/images/search/dental/)

## Usage

Images can be referenced in components using:
```tsx
<Image src="/images/hero-dental-clinic.jpg" alt="Dental Clinic" />
```

Or as CSS backgrounds:
```css
background-image: url('/images/hero-dental-clinic.jpg');
```
